# Finance Android App

A starter Kotlin Android app using Jetpack Compose for finance workflows.

## Stack
- Kotlin
- Jetpack Compose
- Material 3
- Android Gradle Plugin 8.4.2

## Run
1. Open this folder in Android Studio (Hedgehog or newer).
2. Let Gradle sync.
3. Run the `app` configuration on an emulator/device.

## Notes
- `local.properties` is intentionally excluded from git.
- Replace app id/package if you have a team namespace.
